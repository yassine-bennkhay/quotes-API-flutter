
# Quotes

a one page app that consume a JSON file of over 1000+ quotes.
## API Reference

#### Get all quotes
##### endpoint: https://type.fit/api/quotes

```http
  GET /api/quotes
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `text` | `string` | the quote |
| `author` | `string` | the author of the quote |
